# react-shopping-cart
> Implement a simple shopping cart website
#### Environment Require
----
1. npm (version 5.6.0 or above)
2. node (version 7.0.0 or above)

#### Get Start
----
Initialize(install the packages in package.json):
```sh
$ npm install
```
Run server:
```sh
$ node run dev
```
Browser: http://localhost:8080/

#### Testing
```sh
$ npm run test
```
